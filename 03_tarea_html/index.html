Introducción al lenguaje

HTML (siglas de HyperText Markup Language) es el lenguaje de marcado estándar utilizado
para crear y estructurar el contenido de las páginas web. No es un lenguaje de programación,
sino un sistema de etiquetas que le indica al navegador cómo debe mostrar los elementos.

Etiquetas de formato

"< hr >" sirve para crear una línea de división y
"< br >" sirve para realizar un salto de linea"
